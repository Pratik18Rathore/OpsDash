# OpsDash - Internal Operations Dashboard

A production-style internal operations dashboard for managing users, data, and analytics with role-based access.

## Tech Stack

- **Frontend**: React, TypeScript, Redux Toolkit, Tailwind CSS
- **Charts**: Recharts
- **Routing**: React Router

## Features

- User authentication with role-based access control
- Dashboard with analytics and activity charts
- User management (Admin only)
- Profile management

## Getting Started

```sh
# Install dependencies
npm install

# Start the development server
npm run dev
```
